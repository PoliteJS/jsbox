JSBox Concatenated Exercises

IMPORTANT!
All the exercises use the JSBox developement build which is not provided within the GitHub repository.

You need to build by running a simple instruction in your terminal: 

npm install && grunt

> You need NodeJS and GruntJS to be installed on your computer!

01. Variable Declaration

var a = true; 
a === true

02. String Length

var a = 'JSBox';
  • typeof a === 'string'
  • a.length >= 5

03. Use Console Log

var clogSpy = sinon.spy(window.console, "log")
var name = 'jsbox';
console.log(name);
chai.expect(clogSpy.withArgs(name).called).to.be.true
