JSBox use ChaiJS Assertions

Why ChaiJS Assertions?!

ChaiJS is a famous assertion library which can help in writing JSBox's tests.

Vanilla JS:

name.indexOf('jsbox') !== -1

ChaiJS:

expect(name).to.contain('jsbox')

Try it you self:

variable name should contain jsbox
every instruction should end with a ;
var name = 'js'
