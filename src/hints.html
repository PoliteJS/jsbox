JSBox Code Hints Example

Why code hints?!

Code hints are a good method to help the user solving complicated tests.

A test can be made of many conditions and you may want to give suggestions when only a part of the test fails. 

jsbox.hint('hint message', truthCondition);

Hint's message is shown only when the truthCondition is true!

Try it you self:

  1. run the box without any code
  2. try to write b = 1;
  3. try to write a = 1;
  4. try to write var a = 1;
create a variable named a
