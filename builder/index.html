Artifax

An artifax is a piece of Javascript which executes before the user's code in the JSBox.

The normal usage of artifax is to create tests utilities like SinonJS's spies: 

var spy = sinon.spy(window, "alert")

Tests

Tests are the core of JSBox and allows to create front end exercises JSBox.

Write into the code field a Javascript truth expression which will evaluates after user's code.
If all the tests passes then the JSBox is solved and becomes green, else the status of the JSBox turns to "failed" and a redish color is applied.
