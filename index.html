JSBox jQuery Plugin

JSBox is an easy to use jQuery plugin you can use to
embed a Javascript sandbox in any web page.

Solve your first JSBox Exercise

you need to type the Javascript code which is needed in order to fullfill the list of conditions exposed in the top section of the box.

var a = "true"; /** * press "run" to test this code! */
  • declare a variable named a
  • a should be true

JSBox Features

  • HTML5, CSS and Javascript sandbox
  • embed whatever external library for both Javascript and CSS
  • run tests on the Javascript code
  • great console output support
  • error handling support
  • run asyncronous code
  • native supporto for ACE editor
  • full event support for a box outcomes and usage
  • fully customizable components
